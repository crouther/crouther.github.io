Custom Server

Server Hardware

  • Google Search Appliance (U2, 2009)
  • CPU: Intel® Xeon® Processor E5620
  • GPU: AMD Radeon RX480 4GB
  • Memory: 16 GB 1366MHz DDR3
  • Storage: 1TB HDD

The software stack is an Ubuntu 18.04 system with minor application and services intances running on my home network to adminstrate and manage multiple computers from platforms varying from Solaris to MacOS to Windows.
